# 🌍 Odyssey Atlas — Tourist Website

Odyssey Atlas is a simple and modern tourist website that helps users explore destinations across Pakistan, Asia, and the world. It is designed to provide an easy and smooth experience for discovering new places.

---

## ✨ Features

- 🌎 Explore destinations by category (Pakistan, Asia, World)
- 🔍 Search destinations بسهولة (easy search)
- 📱 Responsive design (works on mobile & desktop)
- 🖼️ Clean layout with images and travel details

---

## 🚀 How to Use

1. Download or clone the project  
2. Open `odyssey-atlas-OPEN-THIS.html` in your browser  
3. Start exploring destinations
